## 📌 Project Overview

This repository contains my Week 1 Cyber Security Internship assignment on **Man-in-the-Middle (MITM) attacks**.

The report explains the concept of MITM attacks, how they work, the risks they pose, and how **HTTPS, SSL/TLS certificates, and encryption** protect users from these attacks.

This project is created for educational purposes only and does not include any attack demonstrations or hacking techniques.
